Proyectos Destacados

  1. Sistema de compra y venta de vehículos - Simulación de compra y venta de autos, caminionetas y motocicletas, con funcionalidades como listas de deseados y filtros personalizados. Desarrollado en Java (sin base de datos real).
  2. Gestión de base de datos de una página web - Interfaz intuitiva inspirada en los datos que almacena la plataforma PetBacker, diseñada para administrar registros y servicios relacionados con el cuidado de mascotas.

  3. Juego de adivinación de animales - Aplicación en Java que usa un árbol AVL para que la máquina adivine en qué animal está pensando el usuario. Incluye un modo alternativo donde el usuario intenta adivinar al animal seleccionado por la máquina.

    4. Para más información sobre mis proyectos y experiencia, visita mi perfil de GitHub.
